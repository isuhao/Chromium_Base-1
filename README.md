# Chromium_Base

It's the 'base' of Chromium, a lot of good stuff in there.
Compiling it into a stand alone library so that it can be used by other projects more easily.

## The 'good stuff's:
 * trace event
 * Threading
 * bind
 * profiler
 * stack trace
 * file utils
 * logging
 * containers
 * json
 * memory tools
 * message loop
 * process tools
 * rand
 * string
 * synchronization
 * task scheduler
 * timer
 * a lot more..

# Build
 * Download and extract Chromium's  depot_tools
 * Build with gn
